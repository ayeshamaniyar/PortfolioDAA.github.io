University Sector Algorithms

Library: Finding Books Using KMP Algorithm

Algorithm Description

The KMP algorithm efficiently finds patterns in book titles by preprocessing the query string to allow quick matching. It avoids redundant comparisons, making it ideal for large datasets.

Sample Output:

Enter the search query: Algorithms
Books matching your query:
- Introduction to Algorithms
- Data Structures and Algorithms

Course Registration: Using Binary Search Tree (BST)

Algorithm Description

The Binary Search Tree (BST) organizes courses in a hierarchical structure, allowing efficient insertion, deletion, and search operations. It ensures no duplicate courses and maintains a sorted order.

Sample Output:

Enter Course ID and Course Name to register:
Course ID: 101
Course Name: Data Structures

Course ID: 102
Course Name: Algorithms

Enter Course ID to search: 101
Course Found: Data Structures

All Registered Courses:
Course ID: 101 | Course Name: Data Structures
Course ID: 102 | Course Name: Algorithms

Grade Tracking: Using Fenwick Tree

Algorithm Description

The Fenwick Tree algorithm is used to track and update cumulative grades efficiently. It supports fast updates and range queries, ideal for managing large datasets of student grades.
